# Polagani Akshay | Interactive 3D Portfolio

This is a premium, high-performance portfolio website built with React, Vite, Framer Motion, and Three.js. It features interactive 3D elements, a code-particle background, and a responsive tech-focused design.

## Features
- **3D Hero Background**: Interactive icosahedron with mouse parallax and scroll tracking.
- **Code Particle Background**: Full-page animated background with floating developer keywords and a neural network mesh.
- **3D Tilt Cards**: Tactile hover effects on projects and skills using spring physics.
- **Cinematic Animations**: Smooth, scroll-triggered section reveals.
- **Vercel Optimized**: Configured for seamless deployment on Vercel.

## Tech Stack
- React 19
- Vite
- Framer Motion
- Three.js
- Lucide React
- Tailwind CSS
